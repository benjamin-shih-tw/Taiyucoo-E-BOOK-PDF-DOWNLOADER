# 📖 Flipbook 電子書下載轉 PDF 工具

一個簡單且強大的 Python 工具，用於下載 Flipbook 格式的線上電子書，並自動轉換為單一 PDF 檔案。

---

## ✨ 功能特色

* 🚀 **自動抓取**

  * 輸入電子書網址，自動偵測總頁數並下載

* 🖼️ **高清優先**

  * 優先下載 `large` 高畫質圖片
  * 若無則自動切換至 `mobile` 版本

* 🧠 **記憶體處理**

  * 全程在記憶體中處理圖片
  * 不產生暫存檔案，乾淨不佔空間

* ⚙️ **高度自訂**

  * 自訂 PDF 檔名
  * 自訂輸出路徑

---

## 🛠️ 安裝教學

請先確認已安裝：

* Python 3.x
* pip

### 1️⃣ 複製專案

```bash
git clone https://github.com/benjamin-shih-tw/-PDF-.git
cd -PDF-
```

### 2️⃣ 安裝套件

```bash
pip3 install requests pillow
```

---

## 🚀 使用方式

### 1️⃣ 開啟終端機

### 2️⃣ 進入專案資料夾

```bash
cd ~/EbookDownloader
```

### 3️⃣ 執行程式

```bash
python3 download_ebook.py
```

### 4️⃣ 輸入資訊

依照提示輸入：

* 📌 **電子書網址**

  * 範例：`index.html` 頁面連結

* 📄 **輸出檔名**

  * 範例：`生物講義.pdf`

* 📂 **儲存路徑**

  * 直接按 Enter → 存在目前資料夾
